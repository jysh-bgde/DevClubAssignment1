# GitAss
Repository for Task 2 of Git Assignment

## How to use this repo?

1. Fork this repo.
2. Clone the fork
3. Add your name
4. Commit and Push to your fork.
5. Create a Pull Request on this repo.


## Submission Format

Try to keep the submission format in name.txt as follows
